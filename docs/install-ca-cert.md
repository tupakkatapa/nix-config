
# Install CA Certificate on Android

Download from `cert.coditon.com`.

## Install Certificate

1. **Settings** > **Security & privacy** > **More security settings** > **Encryption & credentials**
2. **Install a certificate** > **CA certificate**
3. Select the `.cer` file

## Firefox

1. Settings > About Firefox > tap logo 5x (enable debug menu)
2. Settings > Secret Settings > "Use third party CA certificates" > enable
